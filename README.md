# Saku

Saku is a financial management application developed collaboratively as our team’s first project. The application is designed to help users manage personal finances efficiently and intuitively.

Tech Stack
Backend: Laravel
Frontend: Flutter
UI/UX Design: Figma
Features
Income management
Expense tracking
Loan and debt management
Multiple wallet support
Custom financial categories
Add and manage wallets
Add and manage categories
Purpose

Saku was created to provide a simple and organized way to monitor financial activities in one application while also becoming an important milestone for our team’s first collaborative development project.

Team

This project was built collaboratively by developers and designers working together to create a functional and user-friendly financial management application.
